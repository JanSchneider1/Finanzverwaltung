Mit diesem Tool können Sie Ihre Finanzen im überblick behalten und Analysieren


Um eine Transaktion hinzuzufügen gehen Sie auf den Reiter "Buchungen". Im unteren Teil der Seite haben Sie die Möglichkeit eine neue Transaktion anzulegen.
Wählen Sie dazu das Datum, tragen Sie einen Namen ein, Ordnen Sie die Transaktion einer Kategorie zu, bestimmen Sie ob es sich um eine Einnahme oder Ausgabe handelt,
geben Sie den Wert an und bestätigen mit einem Klick auf den Haken.

Im oberen Teil der Buchungsseite können Sie Filter eingeben und sich so eine Liste von Buchungen anzeigen lassen, welche auf die Filter passt.



Um eigene Kategorien hinzuzufügen gehen Sie auf den Reiter "Kategorien".
Geben Sie im unteren Teil der Seite einen neuen Kategorienamen ein und bestätigen Sie Ihre eingabe mit einem klick auf den Haken.

Sie können auch den Namen verhandener Kategorien ändenr, indem Sie in das jeweilige Feld einen neuen Namen eingeben und dann auf den Haken klicken



Dieses Programm bietet Ihnen ebenfalls die Möglichkeit Buchungen automatisch zu erstellen. So können Sie zum Beispiel einen Monatlichen Dauerauftrag abbilden.

Klicken Sie dazu auf den Reiter "Fixa". Nun könen Sie im unteren Teil der Seite eine Buchung eintragen, welche Sie mit einem Klick auf den Haken bestätigen.
Diese Buchung wird ab dem Startdatum regelmäßig in der vorgebenen Frequenz getätigt.
Sollte sich das Startdatum in der Vergangenheit befinden, so werden sämtliche Buchungen bis zum heutigen Datum nachgetragen.
